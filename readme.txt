Use the simple shortcode [tweet] to add a link on a phrase and click on the link share it on Twitter! Options and style included.

== Description ==
Using the shortcode [tweet] ... [/tweet] allow your site users to click on a link that post the content of the shortcode on twitter. Really simple to use, with a modern style for the links and some options: text or symbols before and after the quote, twitter username  used for "via" and include permalink.

== Installation ==
Just download the plugin and put it on the /wp-content/plugins directory of you Wordpress installation.

1. Download the .zip file from the official repository and unzip it
2. Upload the folder `tweet-phrase` to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Configure it and...
5. Done!
